## Rectangle & Circle 

### Rectangle 
e.g.) canvas.drawRect(100,100, 200,200, Pnt); <br>
`앞 2개의 x,y좌표`에서 `뒤2개의 x,y좌표`까지 색칠! (드래그하는 것 처럼)

### Circle
e.g.) canvas.drawCircle(50, 50, 40, Pnt); <br>
'앞 2개의 x,y좌표`가 `원의 중심 좌표`이다. <br>
3번째 parameter는 반지름의 크기이다! <br><br>
Circle의 테두리(외곽선) 특징은 코드의 주석을 보고 참고하며 떠올리기!
